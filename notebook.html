How to Use

  • Add Notes: Return to the dictionary, search for a word, and click the "Add Note" option to create a study note.
  • Search: Use the search box above to filter through all your notes by word or content.
  • Review: Revisit your notes anytime to reinforce vocabulary learning.
  • Delete: Remove notes you no longer need by clicking the delete button next to each entry.
  • Backup: Export all your notes using the Backup & Restore page to save them securely.
